# Maze solver
Solves `./maze.png` and outputs it to `./solved.png` using Dijkstra's algorithm and PIL

## Example
![alt Example](example.png)
Here, `maze.png` and `solved.png` are both 5px tall and 5px wide

## Colors
- **White** (#FFFFFF): Paths
- **Black** (#000000): Walls
- **Red** (#FF0000): Start
- **Green** (#00FF00): End
- **Blue** (#0000FF): Solution
